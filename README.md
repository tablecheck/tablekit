# tablekit

Next generation UI component library

## Installation

You will require [pnpm](https://pnpm.io/installation) and [rushjs](https://rushjs.io/pages/developer/new_developer/).

```
nvm use
rush install
```

With these you can use the subfolders as if they were separate repositories.

## Publishing Packages

Publishing packages is handled automatically via GitHub Actions
