# Chips WIP

<IntroText>
  Chips allow users to enter information, make selections, filter content, or
  trigger actions. While buttons are expected to appear consistently and with
  familiar calls to action, chips should appear dynamically as a group of
  multiple interactive elements.
</IntroText>

#### Figma: [Chips](<https://www.figma.com/file/mTd42WpVg1Ehjl3GImVKNS/%5BPublic%5D-TableKit-Components-(3.0)?node-id=4006%3A51372>)

## [Variants](#variants)

Chips have two variants - text only, and text with leading icon.

![Title text](/static/img/components/chips-varients.png 'IMAGEDESCRIPTION')

## [Anatomy](website/src/design/components/chips.html#anatomy)

The fixed height of the chip component is 48px, with vertical padding of 12px and horizontal padding of 16px.

There are text-only, and text with icon variants.

The standard font size inside the buttons is 16px IBM Plex with a medium font-weight.

Corners should always be 100% rounded, and border is 2px thick in all states.

![Tabs anatomy](/static/img/components/chip-anatomy.png 'IMAGEDESCRIPTION')
