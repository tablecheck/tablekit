## [Anatomy](website/src/design/components/tabs.html#anatomy)

The default total size of the button component is 48px, but depending on the state the padding will be different.

Standard (default, hover, active etc) will use 12px padding on the top and bottom, along with 16px on each side.

Active will have 8px padding bottom, with a 4px border.

The standard font size inside the buttons is 16px IBM plex with a medium font weight.

Hover and active states will have a 4px border radius.
