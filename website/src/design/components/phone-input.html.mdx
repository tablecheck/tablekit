## [Anatomy](website/src/design/components/phone-input.html#anatomy)

Similar to the input component, the height of the phone input is 48px, not including external label, character count, or helper/error messages.

Horizontal padding should always be 16px, and vertical 12px.

The country flag is 28 x 20 pixels and is separated from the country code by 4 pixels.

Horizontal padding between items within the input should be 8px.

All variants have a 4px border-radius.
