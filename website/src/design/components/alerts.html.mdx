# Alerts

<IntroText>
  Alerts to notify the user of events, communication actions that are required
  or that have taken place.
</IntroText>

[Figma](<https://www.figma.com/file/W3hxWj6VZydNaNGJSGxRsV/TableKit-Components-(3.0)?node-id=1545%3A4575>)

## [Varients](website/src/design/components/badges.html#varients)

Alerts come can be used with different sentiments to reflect the message than is being communicated. There is a standard size containing the title with content, and an optional icon, or a smaller version with no title.
