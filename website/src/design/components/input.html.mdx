## [Anatomy](website/src/design/components/input.html#anatomy)

The height of all inputs is 48px, not including external label, character count, or helper/error messages.

Horizontal padding should always be 16px, and vertical 12px (except for the “thin” variant’s active state which should have 6px, to make space for the internal label).

There are two optional spots for icons each to the side of the text.

Horizontal padding between items within the input should be 8px.

All variants have a 4px border radius.
