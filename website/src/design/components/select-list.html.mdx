## [Anatomy](website/src/design/components/select-list.html#anatomy)

The default total height _ width of the checkbox component is 24px _ 24px, no vertical or horizontal padding is required within the component structure.

The checkbox square has a 1px border when in an unchecked state, and filled with a checkmark icon when in a checked state.

The font size inside the buttons is 16px IBM Plex with a regular font-weight.

The red asterisks are optional if the field is required.
