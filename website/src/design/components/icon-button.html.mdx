# Icon button WIP

<IntroText>
  A icon button is a icon which triggers an action on click, they communicate
  calls to action to the user and allow users to interact with pages in a
  variety of ways
</IntroText>

#### Figma:[Icon button](<https://www.figma.com/file/W3hxWj6VZydNaNGJSGxRsV/TableKit-Components-(3.0)>)

## [Variants](#variants)

Each variant of the icon button component has its own use-case for both establishing hierarchy, and improving usability, their use-cases will typically follow the same as standard buttons.

## [Anatomy](website/src/design/components/icon-button.html#anatomy)

The total default size of the button component is 48px, composed of an inner carbon icon with a size of 24px and 12px padding on all sides.
