## [Anatomy](website/src/design/components/radio-button.html#anatomy)

The default total height of the radio component is 24px, no vertical or horizontal padding is required within the component structure.

The radio button icon should have a height _ width of 24px _ 24px.

The font size inside the buttons is 16px IBM Plex with a regular font-weight.

The red asterisks are optional if the field is required.
