# Calendar & date picker WIP

<IntroText>
  Date pickers and Time pickers allow selecting a single value from a
  pre-determined set.
</IntroText>

## [Variants](#variants)

Here is where the variant text goes

![Title text](/static/img/components/date-picker-varients.png 'IMAGEDESCRIPTION')

**Single Date:**
Single date should be used when a single date selection is needed.

**Range:**
Date range should be used when a start and end date need to be defined.

## [Anatomy](website/src/design/components/calendar-date-picker.html#anatomy)

The date picker is made up of 3 sub-components within a container, these are the month/year selector, the day column headings, and the date squares, which are displayed in a grid.

![Title text](/static/img/components/calendar-anatomy-1.svg 'IMAGEDESCRIPTION')![Title text](/static/img/components/calendar-anatomy-2.svg 'IMAGEDESCRIPTION')
