# Button group WIP

<IntroText>
  Button groups are used to select or toggle between different grouped options
  where there are a fixed number. They should not be used when the number of
  options are variable or need to scale indefinitely.
</IntroText>

#### Figma: [Button group](<https://www.figma.com/file/W3hxWj6VZydNaNGJSGxRsV/TableKit-Components-(3.0)?node-id=2259%3A74833>)

## [Variants](#variants)

While there are no real different varients, the button group component can be used as an icon button, a text button or a icon + text button. Note: Please don't mix these types from within a button group instance.

![Title text](/static/img/components/button-group-varients.png 'IMAGEDESCRIPTION')

## [Anatomy](website/src/design/components/button-group.html#anatomy)

The height of the button group is 48px, horizontal padding is 16px, vertical padding is 12px, spacing between icon & text is 8px.

To maintain balance, icon size changes depending on variant - with text should be 20px*20px, icon only is 24px*24px.

The standard font size inside the buttons is 16px IBM plex with a medium font-weight.

Border radius is 4px, same as on standard buttons.

![Tabs anatomy](/static/img/components/button-group-anatomy.svg 'IMAGEDESCRIPTION')
